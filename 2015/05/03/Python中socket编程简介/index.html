几点说明：

  1. bind((host, port))：host是客户端连接本socket时需要使用的ip地址，一个主机上可能有多个ip地址，比如最简单的127.0.0.1与192.168.16.122都是指向本机的IP地址，如果绑定的是127.0.0.1，其他人就不能连接到你的机器上了，如果绑定的是192.168.16.122，同一个局域网中的其他用户也可以连接当前的机器上。如果host为””或”0.0.0.0”，则可以监听该机器上所有的ip地址。
  2. listen / recv 函数会使当前程序阻塞
