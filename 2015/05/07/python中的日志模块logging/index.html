python中的日志模块logging

By djskl

发表于 2015-05-07

python中的日志分为4个模块：logger, handler, filter, formatter，其中formater决定了日志的输出格式，filter对日志输出类型进行过滤，handler决定日志的输出目的地，以上3个不同组合可以构成不同的logger，日志在以上4个模块之间传递的是LogRecord对象。
