Django模板标签if与过滤标签default

By djskl

发表于 2015-11-19

这段时间好久不用Django中的模板标签了，乍一用感觉if与default有点乱，特此整理一下：

  • if标签：除is外,支持==, not, !=, True/False的判断与python相同。

还有2个ifequal, ifnotequal标签，效果等同于if var == val, if var != val，而且在后续版本中，这个两个标签会被屏弃。

  • var|default:”def”：此处官方文档是这么说的：

    If value evaluates to False, uses the given default. Otherwise, uses the value.

如果只针对None进行判断，可以使用var|default_if_none:"def"
