By djskl

发表于 2016-04-24

1、安装mod_wsgi

The mod_wsgi package provides an Apache module that implements a WSGI compliant interface for hosting Python based web applications on top of the Apache web server.【引自：MOD_WSGI overview】

安装可以通过官网提供的方法。在Fedora系统上，可直接通过yum安装。

2、编写wsgi脚本
通过werkzeug编写一个简单的脚本，可以取名为myapp.wsgi，放在/var/local/wsgi/scripts目录下。

mod_wsgi.so是一个DSO(Dynamic Shared Objects)文件

DSO提供了一种方法，将一段代码编译成一种特殊格式后可在一个可执行程序运行时将这段程序加载到它的地址空间中【引自：在ApacheHTTPD服务器中使用DSO完全分析】

这样的意思是url中wsgi后面跟啥就访问哪个py文件，比如http://localhost/wsgi/hello，对应的py文件是scripts目录下的`hello`。(When this is used, the next part of the URL after the URL prefix is used to identify which WSGI application script file within the target directory should be used. Both the mount point and the directory path must have a trailing slash）
