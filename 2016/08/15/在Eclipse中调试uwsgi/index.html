在Eclipse中调试uwsgi

By djskl

发表于 2016-08-15

1、将pydevd.py所在目录添加到sys.path

注：此步骤必须使用pydevd，不可使用pdb

2、打开eclipse中的debugger server
点击右图红圈中的按钮即可(如果工具栏没有可以通过Window > Customize perspective > Tool Bar visibility > PyDev debug启用)，此时可在console面板中看到以下输出：

Debug Server at port: 5678

3、启动uwsgi程序
启动uwsgi程序，并通过http访问uwsgi中被调试的接口，会发现程序会阻塞在settrace()语句上，同时可以打开Debug Perspective进行相关的debug操作。

以上同样适用于调试其他不经过eclipse启动的远程程序。
